SK Scoring Cric - Advanced live starter package

Steps to use:
1) Unzip and upload files to your GitHub repo root (or copy contents).
2) Open firebase-config.js and replace placeholders with your Firebase project's config.
3) In Firebase Console enable Authentication (Email/Password and Google). Add your site domain to Authorized domains.
4) Create a Realtime Database and ensure databaseURL matches firebase-config.js.
5) Deploy to GitHub Pages (main branch, root). Open index.html -> Signup/Login -> Dashboard.
6) Use Create pages or quick "My ..." pages to add teams/players/tournaments. They auto-save and show live.
